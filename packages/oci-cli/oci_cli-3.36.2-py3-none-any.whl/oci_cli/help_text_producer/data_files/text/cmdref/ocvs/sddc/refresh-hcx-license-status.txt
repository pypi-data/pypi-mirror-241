Description
===========

Refresh HCX on-premise licenses status of the specified SDDC.


Usage
=====

   oci ocvs sddc refresh-hcx-license-status [OPTIONS]


Required Parameters
===================

--sddc-id [text]

The OCID of the SDDC.
